# splitwise-like


Cost splitting tool similarly to Splitwise App

Workflow
1. Add user
2. Add expense
3. Calculate Payment, who need to pay who. 


## Quick start

### 1. install dependencies
```bash
pip install -r requirements.txt
```

### 2. start the server
```bash
uvicorn main:app --reload
```

### 3. start UI
Open  http://127.0.0.1:8000 for UI.
